# Contributing to @namespacelabs/actions-toolkit

Thank you for your interest in contributing!

## Prerequisites

- Node.js 20+
- [pnpm](https://pnpm.io/)

## Setup

```bash
pnpm install
```

## Development

```bash
# Type check
pnpm run typecheck

# Lint
pnpm run lint

# Format
pnpm run format

# Run tests
pnpm run test
```

## Code Style

Please ensure your code passes linting and formatting before submitting a PR:

```bash
pnpm run lint
pnpm run format:check
```

## Submitting Changes

Before submitting a pull request, please [open an issue](https://github.com/namespacelabs/nscloud-actions-toolkit/issues/new) describing the problem you're solving or the feature you'd like to add.

1. Open an issue explaining the problem or feature
2. Fork the repository
3. Create a feature branch
4. Make your changes
5. Run tests and linting
6. Submit a pull request linking to the issue
